Documentação do projeto: Sistema de Gerenciamento ERP

Este projeto é um ERP (Enterprise Resource Planning), ou Sistema de Planejamento de Recursos Empresariais, é um software que integra diversas funções e processos de uma empresa em uma única plataforma. Ele facilita a gestão e automação de atividades diárias, como finanças, recursos humanos, produção, logística, vendas, compras, etc., promovendo uma visão global e integrada dos dados empresariais.
Para este projeto vamos integrar 6 módulos:

  • Administração de redes locais (0801)
  • Processamento computacional (0802)
  • Programação em C/C++ avançada (0810)
  • Administração de bases de dados para programadores (3933)
  • Programação de sistemas distribuídos - JAVA (0816)
  • Programação avançada com Python (10794)

    • Para este trabalho, vamos utilizar as linguagens: C/C++, Python, Java
